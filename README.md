# Keen Keeper

## 📋 Description

Keen Keeper is a full-featured social companion app designed to help you manage your friendships effectively. Whether you want to keep track of your friends' information, visualize your social network statistics, or maintain a timeline of important moments, Keen Keeper has you covered with a sleek, user-friendly interface.

## 🛠️ Technologies Used

- **Frontend Framework:** [Next.js](https://nextjs.org/) (16.2.3) - React framework for production
- **UI Library:** [React](https://react.dev/) (19.2.4) - JavaScript library for building interfaces
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (4.2.2) - Utility-first CSS framework
- **Component Library:** [DaisyUI](https://daisyui.com/) (5.5.19) - Component library for Tailwind CSS
- **Data Visualization:** [Recharts](https://recharts.org/) (3.8.1) - Composable charting library
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/) (5.6.0) - Popular icon sets
- **Notifications:** [React Toastify](https://fkhadra.github.io/react-toastify/) (11.0.5) - Toast notifications
- **Loading State:** [React Loader Spinner](https://github.com/mhnpd/react-loader-spinner) (8.0.2)

## ⭐ Key Features

1. **Friend Management** - Add, view, and organize your friends with an intuitive friend management interface. Easily maintain a comprehensive list of your social connections.

2. **Statistics Dashboard** - Visualize your friend network with interactive charts and analytics. Track key metrics and gain insights into your social circle at a glance.

3. **Timeline View** - Chronicle important moments and milestones in your friendships. Keep a record of special events and memories with your friends.
